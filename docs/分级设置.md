

# 卵巢-附件超声报告和数据系统 (O-RADS) 风险分层

## 表1 卵巢-附件超声报告和数据系统的风险分层

| O-RADS 评分 | 风险分层 | 术语描述 |
| :--- | :--- | :--- |
| **0** | 评价不完全 | 不适用 |
| **1** | 绝经前正常卵巢 | 1. ≤3cm 的单纯性囊肿<br>2. 黄体囊肿 ≤3cm |
| **2** | 几乎可以肯定的良性病变（＜1%恶性风险） | 1. **单纯性囊肿：**<br>   - ≤3cm<br>   - ＞3cm，≤5cm<br>   - ＞5cm，＜10cm<br>2. **典型良性病变（≤10cm）：** 见表2描述术语<br>3. **囊内壁光滑的非单纯单房囊肿：**<br>   - ≤3cm<br>   - ＞3cm，＜10cm |
| **3** | 恶性低风险病变（1%~10%恶性风险） | 1. **典型良性病变（>10cm）：** 见表3（由医生选择）<br>2. **单房囊肿 ≥10cm**（单纯或非单纯性囊肿）<br>3. **典型病变 ≥10cm：** 包括典型黄体囊肿、卵巢子宫内膜异位囊肿及卵巢出血性囊肿<br>4. **不规则内壁单房囊肿：** 有厚度 ＜3mm 的不规则内壁，任意大小<br>5. **多房囊肿：** ＜10cm，光滑内壁，彩色血流评分 1~3 分<br>6. **实性病变：** 光滑，任意大小，彩色血流评分 1 分 |
| **4** | 恶性中等风险病变（10%~50%恶性风险） | 1. **多房囊肿，没有实性成分：**<br>   - ≥10cm，内壁光滑，彩色血流评分 1~3 分<br>   - 任意大小，内壁光滑，彩色血流评分 4 分<br>   - 任意大小，内壁不规整和（或）分隔不规则，任意彩色血流评分<br>2. **有实性成分的单房囊肿：** 任意大小，0~3 个乳头样突起，任意彩色血流评分<br>3. **有实性成分的多房囊肿：** 任意大小，彩色血流评分 1~2 分<br>4. **实性肿物：** 光滑，任意大小，彩色血流评分 2~3 分 |
| **5** | 恶性高风险（≥50%恶性风险） | 1. **单房囊肿：** 任意大小，有 ≥4 个乳头状突起，任意血流评分<br>2. **有实性成分的多房囊肿：** 任意大小，彩色血流评分 3~4 分<br>3. **实性肿物：**<br>   - 光滑，任意大小，彩色血流评分 4 分<br>   - 不规整，任意大小，任意血流评分<br>4. **腹水积液，有或无腹膜结节** |
